About TGS and WebP Formats

                    
TGS (Telegram Sticker): A compressed format for animations based on Lottie JSON, optimized for small file sizes. Perfect for lightweight applications and supported by the LottieFiles TGS player.

                    
WebP: A modern image format developed by Google that supports animation with smaller file sizes compared to GIFs. It offers better compression and quality for animated content on the web.

                    
Both formats in this library are animated, but they serve different use cases. TGS is ideal for applications that support Lottie animations, while WebP works everywhere without additional libraries.
